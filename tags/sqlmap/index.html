    SQLMap是一个自动化的SQL注入工具，它的主要功能是扫描，内置了很多绕过插件，支持的数据库有MySQL、Oracle、PostgreSQL、Microsoft SQL Server、Microsoft Access、IBM DB2、SQLite、Firebird、Sybase和SAP Max DB。其强大功能有数据库指纹识别、数据库枚举、数据提取、访问目标文件系统。
